# History

This folder archives earlier frameworks that informed the current model.

The repository originally used the CATS framework, later extended to CATSS when Social Action was introduced as a fifth pillar:

- CATS: Capability, Autonomy, Tutoring, Stewardship
- CATSS: Capability, Autonomy, Tutoring, Stewardship, Social Action

The current operational framework is [STACK](../how/README.md): Stewardship, Tutoring, Autonomy, Capability, Kinship.

Social Action was replaced by Kinship to better reflect the relational and fellowship-oriented nature of that pillar.
